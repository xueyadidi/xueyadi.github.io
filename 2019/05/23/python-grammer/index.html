这一段时间在用Python搞图像处理，说实话，上一次学习Python还是两年前，使用的是《Learn Python The Hard Way》(笨方法学python)。这本书的语法是基于Python2的，两年前Python2和Python3还是占据一样的粉丝，但到今年（2019），很明显Python2大势已去了。悔不当初呀，当时直接选Python3就好了，可以好好反思一下因循守旧和保持开放性这个对立面问题了。。。（又上升到哲学qwq，我的心好累）。。

写这篇文章的主旨，就是简单学习记忆一下Python3的语法知识，教程参考的是菜鸟教程，感觉这个网站文章也是用markdown写得哈哈。
